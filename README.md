# Requirements

    ## Node Version
        16 | 18 | 20
    ## OS
        Windows 10 | Mac OS
    ## Browser
        Chrome | Edge

# Running instructions

- Copy project to your directory ( `git clone https://github.com/VENUS-git1/Yield-Farming-DApp.git` )
- Install project packages (`npm install --legacy-peer-deps`)
- Start project (`npm run start`)
